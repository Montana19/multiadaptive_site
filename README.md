# Многостраничный адаптивный сайт

Корпоративный сайт с адаптивной вёрсткой. Корректно отображается на мобильных устройствах, планшетах и десктопах — от 320px до 1920px.

## 🔍 Особенности

- Адаптивная вёрстка
- Три разрешения: мобильные (320–767px), планшеты (768–1279px), десктоп (1280px+)
- Flexbox и CSS Grid для построения сеток
- Плавная адаптивность, без горизонтальных скроллов

## 🛠 Технологии

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Figma (чтение макета)
